條件組合
True and True 是 True
True and False 是 False
True or True 是 True
True or False 是 True
False or False 是 False
False and False 是 False

我們這邊用了float()這個函數,這個函數是把整數和字串換成浮點數 白話來說就是把 1 換成1.0。
